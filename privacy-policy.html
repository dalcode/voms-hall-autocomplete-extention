개인정보처리방침

최종 수정일: 2025년 6월 13일

본 개인정보처리방침은 VOMS 웨딩홀 자동완성 Chrome 확장프로그램(이하 "확장프로그램")이 수집·저장·사용하는 정보에 대해 설명합니다.

1. 수집하는 정보

항목 저장 위치 목적
VOMS API URL Chrome 동기화 스토리지 (chrome.storage.sync) 웨딩홀 목록 API 서버 주소 설정
VOMS API 키 Chrome 동기화 스토리지 (chrome.storage.sync) API 인증
웨딩홀 목록 캐시 Chrome 로컬 스토리지 (chrome.storage.local) 자동완성 후보 목록 로컬 캐시 (24시간 갱신)

2. 수집하지 않는 정보

본 확장프로그램은 다음 정보를 수집하지 않습니다.

  • 사용자의 개인 식별 정보 (이름, 이메일, 전화번호 등)
  • 브라우저 방문 기록 또는 검색 내역
  • 입력 폼의 내용 (자동완성 선택값 제외)
  • 위치 정보

3. 정보의 외부 전송

확장프로그램은 설정된 VOMS API 서버에만 웨딩홀 목록 조회 요청을 전송합니다. 수집된 정보는 제3자에게 판매·공유·제공되지 않습니다.

4. 정보의 보관 및 삭제

저장된 정보는 Chrome 브라우저의 스토리지에만 보관되며, 확장프로그램을 삭제하거나 Chrome 설정에서 스토리지를 초기화하면 즉시 삭제됩니다.

5. 대상 사용자

본 확장프로그램은 VOMS 시스템을 사용하는 사내 임직원 전용으로 제작되었으며, 일반 공개 서비스가 아닙니다.
